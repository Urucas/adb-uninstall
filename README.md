# adb-uninstall [![Build Status](https://travis-ci.org/Urucas/adb-uninstall.svg?branch=master)](https://travis-ci.org/Urucas/adb-uninstall)
Uninstalling Android packages from cli

#Install
```bash
npm install -g adb-uninstall
```
